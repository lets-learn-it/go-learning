# loops

## For loop

```go
for /* init */ i := 0; /* condition */ i < 10; /* post */ i++ {
    // do things
}
```

- *init* and *post* are optional
- *while* kind of
  ```go
  i := 1
  for i < 10 {
    // do thing
    i += 1;
  }
  ```
- *infinite loop*
  ```go
  for {
    // do things forever
  }
  ```
